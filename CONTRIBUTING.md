# Contributing to CPFA (Climate Prediction For All)

Thank you for your interest in contributing to **CPFA**!  
Our goal is to make climate prediction and visualization easier for students and non-experts.

## How to Contribute
1. **Fork** this repository to your own GitHub account.  
2. **Create a new branch** for your feature or bug fix.  
3. **Commit** your changes with a clear and descriptive message.  
4. **Submit a Pull Request (PR)** for review and approval.  

Before contributing, please ensure that your work follows the **Apache 2.0 License**.

## Code Style
- Follow PEP8 for Python code formatting.  
- Keep comments clear and concise.  
- Use English for variable names, comments, and commit messages.

## Reporting Issues
If you find a bug or have an idea for improvement, please open an issue on GitHub with a detailed description.

We appreciate your contributions to make **CPFA** better for everyone!
